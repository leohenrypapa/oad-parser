# Legacy Artifact Map

## Purpose

This note records that historical legacy parser materials informed the foundation extraction and compatibility direction.

## Customer-safe status

Legacy trees, deployment files, sample outputs, private captures, and local operator directories are not included in the customer source pack.

Historical references are not authoritative by themselves. Use current code, tests, traceability docs, and approved references as the release baseline.

## Compatibility rule

Legacy behavior may be used for regression comparison where approved source material exists. It must not be used to invent or assert new radar message semantics without approved sanitized captures or authoritative message-format references.
